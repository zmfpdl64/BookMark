## 코드 작성 스타일

DDD를 사용하지 않고 서비스 주도 설계로 코드를 작성하는 이유
초기 개발이기 떄문에 변경으로부터 영향이 적어야 한다고 생각했으며 
외래키를 사용하지 않는 정책에 따라 Lazy Loading과 Eager Loading을 사용할 수 없으면 직접 Dto를 사용하여 맵핑할 것입니다.
